OniPlayer Controls
====================

Keyboard Controls
---------------
Playback:
- Space: Play/Pause
- Enter: Toggle fullscreen
- F: Toggle fullscreen
- Left Arrow: Seek backward
- Right Arrow: Seek forward
- Up Arrow: Volume up
- Down Arrow: Volume down
- M: Toggle mute
- Esc: Exit fullscreen

Mouse Controls
------------
Playback:
- Double-click: Play/Pause
- Middle-click: Toggle fullscreen
- Right-click: Show context menu
- Scroll wheel: Adjust volume
- Left click + scroll: Fast forward/rewind
- Click and drag timeline: Seek through video
